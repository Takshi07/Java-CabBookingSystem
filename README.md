# 🚖 Java Cab Booking System

A **Java Swing–based mini project** that provides a complete cab-booking solution with modules for **Cab Management**, **Booking**, **Analysis**, and **Data Export**.

---

## 🎯 Objective
To design and implement a **user-friendly Cab Booking System** that enables efficient cab management, booking, data analysis, and export through an interactive graphical interface built in **Java Swing**.

---

## 🧩 Modules Overview

### 1️⃣ Cab Management
- Add, edit, or delete cab details (ID, driver name, status, etc.)
- Maintain availability status in real time.

### 2️⃣ Booking
- Allows customers to book available cabs.
- Automatically updates cab status and booking records.

### 3️⃣ Analysis
- Displays total trips, revenue generated, and performance statistics.
- Visual summaries for admin review.

### 4️⃣ Export
- Exports cab and booking data to CSV or text files for future reference.

## 🧠 OOP Concepts Implemented
- **Encapsulation:** Data hidden using private fields with getters/setters.  
- **Inheritance:** Subclasses for different modules sharing common behavior.  
- **Polymorphism:** Method overriding for flexible booking and cab actions.  
- **Abstraction:** Separate classes for GUI, logic, and data handling.
