# Java Cab Booking System

A **Java Swing–based mini project** that provides a complete cab-booking solution with modules for **Cab Management**, **Booking**, **Analysis**, and **Data Export**.

---

## Modules 

### Cab Management
- Add, edit, or delete cab details (ID, driver name, status, etc.)
- Maintain availability status in real time.

### Booking
- Allows customers to book available cabs.
- Automatically updates cab status and booking records.

### Analysis
- Displays total trips, revenue generated, and performance statistics.
- Visual summaries for admin review.

### Export
- Exports cab and booking data to CSV or text files for future reference.
